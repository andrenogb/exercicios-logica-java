# ☕ Fundamentos de Orientação a Objetos com Java

Este projeto foi desenvolvido como parte dos meus estudos práticos de Java. O foco principal foi entender como uma classe molda um objeto e como os métodos interagem com os dados internos.

## 🎯 Objetivos do Exercício
- Criar a classe `Pessoa` com atributos reais (`String`, `int`).
- Implementar o método `olaMundo` para prática de sintaxe inicial.
- Desenvolver o método `exibeFichaPessoa` para manipulação e exibição de atributos.

## 🧠 Conceitos Aprendidos

### A Diferença entre `void` e `return`
Um dos pontos chave deste estudo foi diferenciar quando um método deve apenas realizar uma ação e quando deve devolver um valor:

1. **Método `void` (Ação):** Utilizado em `exibeFichaPessoa()`. Ele "abre o guarda-roupa" e mostra o que tem dentro no console, mas não entrega nenhum dado para ser guardado em variáveis na classe `Main`.
2. **Métodos de Tipo (Retorno):** Compreendi que, para usar um resultado num cálculo futuro, o método deve especificar o tipo (ex: `int`, `double`) e usar a palavra-chave `return`.
